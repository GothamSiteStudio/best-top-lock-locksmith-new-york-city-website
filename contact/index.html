Before You Submit

A better request gets a better first response

If you want a faster estimate, include the borough, whether the job is emergency or scheduled, and what you think the issue is. For example: apartment lockout in Astoria, top lock installation for a Manhattan apartment, or lock change after move-in in Park Slope. Even a short message like that helps route the request correctly.

Some locksmith jobs are simple and can be priced quickly. Others need a little more context because the right answer depends on the current lock, the door prep, or building restrictions. The form is here so customers have a dedicated contact page from the header and a clean place to submit those details.

Helpful details to include

  • The borough and neighborhood.
  • The type of door or building if you know it.
  • Whether this is emergency work or a planned appointment.
  • Whether you think the job is a lockout, rekey, lock change, top lock, smart lock, or commercial issue.
